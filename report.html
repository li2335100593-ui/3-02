輪播 SDK 部署指南

將以下腳本嵌入目標網站，即可實現自動輪播 + 數據追踪

<script src="https://li2335100593.github.io/3-02/carousel.js"></script>

info 內置配置說明

carousel.js v3.0 已內置 URL 列表（livingroom-design / old-house-renovation / incar），無需額外配置。直接引入腳本即可工作。

同域鏈接順延機制

用戶在目標網站 A 中點擊任意同域鏈接時，輪播狀態會自動合併到新鏈接，倒計時順延而非重置。

check_circle 已實現

點擊同域鏈接時，輪播計時器狀態（已過時間、當前頁面索引）會通過 URL hash 傳遞到新頁面，倒計時從當前進度繼續。

warning 注意

跨域鏈接（跳转到其他域名）會導致輪播狀態丟失，計時器重置。建議確保輪播目標頁面都在同一域名下。

Scheduler 調度器使用

播放人員專屬鏈接的格式。調度器會將播放人員標識傳入輪播狀態，後續所有頁面事件都會標記為該人員。

https://li2335100593.github.io/3-02/scheduler.html?u=OP001
  • ?u=OP001 — 播放人員編碼（必填）
  • 調度器讀取配置後自動跳轉到第一個目標 URL
  • 所有後續輪播頁面都會攜帶該人員標識
  • 無需在目標網站手動配置任何參數
